# Agents

One subdirectory per agent. Phase 1 contains no agents yet.

Phase 2 (Telemetry primitives) populates `_lib/runs.py` — the cost-emission
helper that every later agent must use.

Phase 3 adds the first real agent subdirectory: `discord-bot/`.

See `architecture/40-action-layer.md` for the full agent roster and
specifications.
