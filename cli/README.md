# Operator CLI

Command-line helpers for ad-hoc inspection and operations.

Phase 1 is empty.

Phase 2 adds `spend.py` for spend-by-agent queries.
Phase 3 adds `recall.py` for hybrid fact search.
Later phases add `health.py` (system check), `icp.py` (signal inspection).

These are run from the laptop or `agent` account against the live brain.
